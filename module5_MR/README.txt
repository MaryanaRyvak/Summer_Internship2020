Інструктція для домашнього завдання 

Домашнє завдання для модулю № 5 - функції та блоки, складається з 2 завдань.
Кожне завдання містить детальний опис наявного коду, використовує новий матеріал
та має описаний один з можливих алгоритмів виконання програми.


Після завершення роботи слід завантажити ваше завдання у систему для його перевірки викладачем.
